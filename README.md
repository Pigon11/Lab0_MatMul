### 3. Что же было распараллелено?
Вычисление каждого элемента результирующей матрицы C[i][j]

Умножение матриц - это чрезвычайно параллельная задача (тип задач в системах параллельных вычислений, для которых не требуется прилагать больших усилий при разделении на несколько отдельных параллельных задач):
1. Каждый элемент C[i][j] вычисляется независимо от других
2. Нет зависимостей по данным между разными элементами
3. Все вычисления идентичны по структуре

### 4. Как было распараллелено?
Один поток - один элемент.
Запускается N×N потоков (столько же, сколько элементов в матрице).
Каждый поток выполняет скалярное произведение своей строки и столбца.
Потоки организованы в блоки 16×16 для оптимального использования ресурсов GPU

GPU имеет тысячи ядер, которые могут работать одновременно. Запуская N×N потоков, загружаются все доступные вычислительные ресурсы, что даёт значительный выигрыш во времени на больших матрицах.
Тестирование проводилось для матриц размером от 100×100 до 2000×2000.
